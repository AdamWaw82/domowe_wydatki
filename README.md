# Domowe Wydatki

## Opis
Aplikacja Flask do zarządzania wydatkami. Umożliwia dodawanie, edytowanie i usuwanie wydatków.

## Wymagania
- Python 3.6 lub nowszy

## Instalacja

### Krok 1: Sklonuj repozytorium
